
# Mano Maestra

Proyecto Mano Maestra: Plataforma de cursos en linea para el aprendizaje de Lenguaje de Señas Mexicano (LSM).
Construido con Next.js (React) y Tailwind CSS.

## Importante

Esta versión del sistema no requiere del backend de Supabase. En cambio, se usan cookies para emular el uso del sistema. Las Funciones de registro e inicio de sesión no funcionan, solo cargan valores de usuario por defecto, y otras funciones como cambiar datos no hacen nada.

El sistema de reconocimiento de señas se encuentra roto de momento.

## Para empezar

1. Clonar el repositorio.
2. Acceder al directorio del proyecto (cd mano-maestra).
3. Ejecutar `npm install` ó `npm i` para instalar las dependencias.
~~4. Crear archivo .env.local con las siguientes variables locales:~~

4. Ejecutar `npm run dev` para activar entorno de desarrollo.
5. Abrir [http://localhost:3000](http://localhost:3000)
